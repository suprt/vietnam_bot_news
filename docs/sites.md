## Источники новостей (RSS и сайты)

### Основные RSS-ленты

1. **Dat Viet**
   - Сайт: `https://datviet.com`
   - RSS: `https://datviet.com/feed`
   - Описание: горячие новости, оборона, наука и технологии.

2. **Việt Báo**
   - Сайт: `https://vietbao.vn`
   - RSS: `https://vietbao.vn/rss/`
   - Описание: новости, внешняя политика, экономика, право, жизнь, технологии, спорт.

3. **Thanh Niên**
   - Сайт: `https://thanhnien.vn`
   - RSS: `https://thanhnien.vn/rss.html`
   - Описание: бизнес, финансы, развлечения, спорт.

4. **Báo Pháp Luật (PLO)**
   - Сайт: `https://plo.vn`
   - RSS: `https://plo.vn/rss/home.rss`
   - Описание: текущие события, общество, право, безопасность.


5. **VnExpress**
   - Сайт: `https://vnexpress.net`
   - RSS (общество): `https://vnexpress.net/rss`
   - Описание: одна из самых читаемых вьетнамских газет, частые обновления.

6. **VietnamPlus**
   - Сайт: `https://www.vietnamplus.vn`
   - RSS: `https://www.vietnamplus.vn/rss.html`
   - Описание: политика, бизнес, общество, спорт, 24/7 обновление.

### Замечания по использованию

- На первом этапе приоритет — использование RSS-лент.
- Если у конкретного источника нет нужного RSS (по тематикам), возможно:
  - поиск дополнительных RSS-разделов на сайте;
  - либо аккуратный HTML-скрейпинг наиболее релевантных разделов.
- Для каждого источника будут реализованы адаптеры с единым интерфейсом получения данных.
