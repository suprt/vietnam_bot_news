# Анализ RSS-фидов и оптимизация

## Критерии фильтрации

### ❌ Удалены следующие типы фидов:

1. **Технические/служебные:**
   - `http://purl.org/rss` - это не RSS-лента, а стандарт
   - `http://purl.org/rss/1.0/modules/content` - техническая спецификация
   - `https://my.thanhnien.vn/page/login.html?...` - страница входа
   - `https://s1cdn.vnecdn.net/.../1000000.js` - JavaScript файл, не RSS
   - `https://plo.vn/rss.html` - HTML страница, не RSS-лента

2. **Слишком специфичные категории:**
   - Видео-ленты (`/video/`, `/multimedia/`)
   - Подкасты (`/podcast/`)
   - Реклама и объявления (`/rao-vat/`, `/tuyen-dung/`, `/viec-lam/`)
   - Специфичные подкатегории (например, `/ban-can-biet/dich-vu/`, `/ban-doc/la-lanh-dum-la-rach/`)
   - Сезонные/временные разделы (`/nhat-ky-tet-viet/`, `/chao-ngay-moi/`)

3. **Дубликаты:**
   - Если есть `home.rss` и главная страница - оставлен только `home.rss`
   - Если есть общая категория и подкатегории - оставлены основные категории

4. **Не релевантные для новостного дайджеста:**
   - Специфичные разделы для молодежи (`/gioi-tre/`)
   - Очень узкие подкатегории (например, `/kinh-te/lam-giau/`)
   - Разделы про моду (`/thoi-trang-tre/`)
   - Потребительские разделы (`/tieu-dung-thong-minh/`)

## ✅ Оставлены фиды по категориям:

### Основные новостные категории (соответствуют `pipeline.yaml`):

1. **Политика** (`chinh-tri`, `thoi-su`):
   - Главные ленты (`home.rss`, `/rss`)
   - Политические разделы

2. **Экономика и бизнес** (`kinh-te`, `kinh-doanh`):
   - Экономические разделы
   - Бизнес-новости

3. **Общество** (`xa-hoi`, `doi-song`, `thoi-su`):
   - Общественные новости
   - Социальные темы

4. **Право и безопасность** (`phap-luat`):
   - Юридические новости
   - Правоохранительные темы

5. **Технологии и наука** (`cong-nghe`, `khoa-hoc`):
   - Технологические новости
   - Научные открытия

6. **Культура и развлечения** (`van-hoa`, `giai-tri`):
   - Культурные события
   - Развлечения

7. **Спорт** (`the-thao`):
   - Спортивные новости

8. **Здоровье и образ жизни** (`suc-khoe`):
   - Медицинские новости
   - Здоровый образ жизни

9. **Мир (международные новости)** (`the-gioi`, `doi-ngoai`):
   - Международные события
   - Внешняя политика

## Итоговая статистика

- **Исходное количество фидов:** ~350+
- **Оптимизированное количество:** ~45 фидов
- **Сокращение:** ~87%

## Преимущества оптимизации

1. **Меньше запросов:** Быстрее обработка, меньше нагрузка на источники
2. **Меньше дубликатов:** Основные категории покрывают все новости
3. **Лучшее качество:** Только релевантные новостные фиды
4. **Проще поддержка:** Меньше фидов = проще обновлять конфигурацию

## Примечания

- Если в будущем понадобятся дополнительные категории, их можно добавить
- Система автоматически обрабатывает ошибки отдельных фидов, так что нерабочие фиды не остановят весь процесс
- Лимит в 100 статей на фид защищает от обработки слишком старых новостей
